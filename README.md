## Description
Noor Bakery Invoice System is a modern web-based application designed to streamline billing and invoice generation for bakery businesses. It features a sleek, user-friendly interface and allows employees to quickly add items, calculate totals, and download invoices as PDFs. 

The backend is powered by **Java**, implementing three core OOP concepts: **Encapsulation, Inheritance, and Polymorphism**.

The frontend is built using **HTML, Tailwind CSS, and JavaScript**, providing a smooth and responsive user experience. Users can log in, manage invoices, preview receipts, and download them effortlessly.
